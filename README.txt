================================================================
  SkyMark v2.0
  MSFS 2024 VFR 파일럿을 위한 실시간 대시보드 오버레이
================================================================

[ 설치 방법 ]

  1단계 - Python 설치 (이미 있으면 건너뜀)
    https://www.python.org/downloads/
    ※ 반드시 64비트 버전
    ※ 설치 시 "Add Python to PATH" 체크 필수!

  2단계 - INSTALL.bat 더블클릭 (최초 1회만)
    필요한 라이브러리를 자동으로 설치합니다.

  3단계 - 이후엔 START.bat 더블클릭으로 실행
    관리자 권한 불필요!

[ 실행 순서 ]

  1. MSFS 2024 실행
  2. 조종석까지 완전히 로딩
  3. START.bat 더블클릭
  4. 상단에 초록불(●) = 연결 성공!

[ 주요 기능 ]

  바람 정보
    - 실시간 풍향/풍속 나침반 표시
    - 기수 기준 / 진북 기준 전환
    - 정면풍(Headwind) / 크로스윈드 성분 분리 표시
    - ※ 풍향은 바람이 불어오는 방향(FROM)입니다

  비행 데이터
    - 기온(OAT), 밀도고도, 고도 MSL/AGL
    - IAS, TAS, VSI, QNH, RPM
    - 연료/RPM 게이지 바 표시

  항법 정보
    - 출발지/목적지 방향과 거리
    - 현재 위치 좌표 (위도/경도)

  가까운 공항
    - 현재 위치 기준 가까운 공항 8개
    - 방향/거리 실시간 표시
    - Google Earth 바로가기

  관심장소 (POI)
    - 비행 중 원클릭으로 현재 위치 저장
    - 저장된 장소까지 방향/거리 표시
    - Google Earth 연동

[ 패널 구성 ]

  각 패널은 독립적으로 닫고 열 수 있습니다.
  하단 버튼 바에서 원하는 패널만 선택해서 보세요.

[ 주의사항 ]

  - VR 모드에서는 표시되지 않을 수 있습니다
  - 조종석 로딩 전에는 연결되지 않습니다
  - 첫 실행 시 공항 데이터를 인터넷에서 다운로드합니다

[ INSTALLATION ]

  Step 1 - Install Python 64-bit (skip if already installed)
    https://www.python.org/downloads/
    Check "Add Python to PATH" during installation!

  Step 2 - Run INSTALL.bat (first time only)
    Installs required libraries automatically.

  Step 3 - Run START.bat to launch
    No administrator rights required!

[ HOW TO USE ]

  1. Launch MSFS 2024
  2. Wait until cockpit is fully loaded
  3. Double-click START.bat
  4. Green dot (●) = Connected!

[ NOTES ]

  - Wind direction = direction wind is coming FROM
  - Not supported in VR mode
  - Airport data downloaded automatically on first run
